# Smart Trike Assets

## Required Assets
Place the following image files in the assets directory:
- icon.png (1024x1024)
- splash.png (1284x2778)
- adaptive-icon.png (1024x1024)

## Fonts
Font files should be placed in assets/fonts/:
- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf
- Questrial-Regular.ttf

Download fonts from Google Fonts:
- Poppins: https://fonts.google.com/specimen/Poppins
- Questrial: https://fonts.google.com/specimen/Questrial
